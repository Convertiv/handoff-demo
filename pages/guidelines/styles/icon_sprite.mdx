## Where to Find

You can find these mixins in `integration/snippets/mixins/icon-sprite.scss`.

## Basic Icon Sprite
This calls the background image and sets the background repeat to no-repeat.
This gets used in all the other mixins.

```scss title="Set the position of before after elements"
@mixin icon-sprite {
  background-image: url($icon-sprite);
  background-repeat: no-repeat;
  background-size: 250px 250px;
}
```
