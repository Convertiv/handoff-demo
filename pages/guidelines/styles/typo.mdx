## Where to Find

- You can find these mixins in `integration/snippets/mixins/typo.scss`. 
- To see all of the typography style code, see [here](/guidelines/syles/fonts/).

## Common Typography Mixin
This sets the base color and family for all the specific typography mixins.

```scss title="Set color and font family"
// This is common typography mixin
@mixin title-common {
  font-family: $font-barlow-sans;
  color: color('pure-black');
}
```

## Specific Typeograpy Style Mixins
These mixins set the font size, line height, weight, spacing and mobile font.
To see a demo of the typography styles, see [here](/foundations/typography/).

```scss title="Font Style Mixins"

  @include title-common;
  font-size: $h1-font-size-mobile;
  line-height: 1.2;
  font-weight: $font-weight-normal;
  letter-spacing: -0.05em;
  @include media-breakpoint-up(lg) {
    font-size: $h1-font-size;
  }
}
```
